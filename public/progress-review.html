🎯 System Overview

The Golden Question contest system is a Laravel 12 + Vue 3 application that enables users to scan QR codes at dog stations and participate in daily trivia contests. The system includes geolocation tracking, automated winner detection, and complete analytics infrastructure.

📊 Technical Stack

Laravel 12 Vue 3 (Composition API) Inertia.js Filament 4 (Admin) Tailwind CSS Vite SQLite (Dev) MySQL (Production) Laravel Echo + Pusher PHPUnit 12.3

🔧 Technical Workflows

QR Code Scan Flow

  1. User scans QR code on sticker at dog station
  2. System logs: IP address, geolocation (lat/long), user agent, timestamp
  3. Redirects to contest page with active question
  4. Sticker scan count incremented for analytics

Contest Submission Flow

  1. User selects answer (A, B, C, or D)
  2. System validates:
    • Contest is active
    • User is authenticated and age 18+
    • User hasn't submitted today for this question
    • IP hasn't submitted today for this question
    • User hasn't won in last 30 days
  3. Submission saved with timestamp and random tiebreaker (1-1,000,000)
  4. If answer is correct:
    • Check if this is first correct answer (timestamp + tiebreaker)
    • If yes: Create Winner record, dispatch gift card delivery job
    • Lock question (no more winners)

Gift Card Delivery Flow

  1. Winner record created in database
  2. DeliverGiftCardJob dispatched to queue
  3. Job calls GiftCardService->deliverGiftCard()
  4. Service makes Tremendous API call (or mock in dev)
  5. On success:
    • GiftCard record updated with redemption link
    • Status set to "delivered"
    • Winner notification email dispatched
  6. On failure:
    • Job retries with exponential backoff
    • After 5 attempts: marked as permanent failure
    • Delivery log created for audit trail

Winner Notification Flow

  1. Email sent via Laravel's notification system
  2. Contains:
    • Congratulations message
    • Question text that was answered correctly
    • Gift card redemption link (if delivered)
    • Support contact information
  3. Also stored in database notifications table

📈 Analytics & Tracking

  • Every QR scan logged with geolocation for heatmap generation
  • Sticker scan counts for performance analysis
  • Submission tracking (correct vs incorrect answers)
  • User winnings history
  • Gift card delivery success/failure rates
  • IP address tracking for anti-cheat analysis
